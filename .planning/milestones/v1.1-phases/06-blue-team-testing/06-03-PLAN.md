**TEST-07 (no LLM backend):**

Manipulate PATH to ensure `which::which("claude")` and `which::which("copilot")` return Err:
```rust
#[test]
fn test_no_llm_backend_graceful_degradation() {
    // Save original PATH
    let original_path = std::env::var("PATH").unwrap_or_default();
    // Set PATH to empty (or a dir with no executables)
    std::env::set_var("PATH", "/nonexistent");

    let config = Config::default_config();
    let backend = config.detect_backend();

    // Restore PATH
    std::env::set_var("PATH", &original_path);

    assert!(backend.is_none());
}
```

WARNING: Env var manipulation is not thread-safe. Use `#[serial]` or accept the risk since integration tests may run sequentially by default (Rust runs integration test files sequentially, tests within a file in parallel by default). To be safe, mark PATH-modifying tests with a comment noting they must not run in parallel with other PATH-dependent tests. Since this is the only test file modifying PATH, it should be fine.

Also test that App with no backend stays Idle:
```rust
let diff_data = diff::parse(SAMPLE_DIFF);
std::env::set_var("PATH", "/nonexistent");

std::env::set_var("PATH", &original_path);
```
