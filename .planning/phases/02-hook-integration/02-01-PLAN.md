**src/event.rs** — Replace placeholder with async event router:
```rust
use crossterm::event::{Event as CrosstermEvent, EventStream, KeyEventKind};
use futures::StreamExt;
use tokio::signal::unix::{signal, SignalKind};
use tokio::sync::mpsc;
use crate::app::Message;

pub async fn event_loop(tx: mpsc::Sender<Message>) {
    let mut reader = EventStream::new();
    let mut sigusr1 = signal(SignalKind::user_defined1())
        .expect("failed to register SIGUSR1 handler");

    loop {
        tokio::select! {
            Some(Ok(event)) = reader.next() => {
                match event {
                    CrosstermEvent::Key(key) if key.kind == KeyEventKind::Press => {
                        if tx.send(Message::KeyPress(key)).await.is_err() {
                            break;
                        }
                    }
                    CrosstermEvent::Resize(w, h) => {
                        let _ = tx.send(Message::Resize(w, h)).await;
                    }
                    _ => {}
                }
            }
            _ = sigusr1.recv() => {
                let _ = tx.send(Message::RefreshSignal).await;
            }
        }
    }
}
```

**src/app.rs** — Extend Message enum, add Command enum, refactor App:

1. Add new Message variants:
   - `Resize(u16, u16)` — terminal resize
   - `RefreshSignal` — SIGUSR1 received
   - `DebouncedRefresh` — debounce timer expired, time to actually refresh
   - `DiffParsed(DiffData)` — async diff parse completed

2. Add Command enum:
   ```rust
   pub enum Command {
       SpawnDiffParse,
       Quit,
   }
   ```

3. Add to App struct:
   - `pub event_tx: Option<mpsc::Sender<Message>>` — for spawning debounce timers
   - `pub debounce_handle: Option<tokio::task::JoinHandle<()>>` — current debounce timer

4. Change `update()` signature to `pub fn update(&mut self, msg: Message) -> Option<Command>`.

5. Implement debounce logic in update():
   - `RefreshSignal` → abort any existing debounce_handle, spawn a new tokio task that sleeps 500ms then sends `DebouncedRefresh` via event_tx. Return None.
   - `DebouncedRefresh` → set debounce_handle to None, return `Some(Command::SpawnDiffParse)`.
   - `DiffParsed(new_data)` → preserve scroll/collapse state by mapping collapsed NodeId::File(idx) to file paths using current diff_data, then replacing diff_data with new_data, then re-mapping paths back to new indices. Rebuild highlight_cache. Clamp selected_index to new visible_items().len(). Return None.
   - `Resize(w, h)` → update viewport_height. Return None.
   - `KeyPress` → existing handle_key logic. 'q' returns `Some(Command::Quit)`.
   - `Quit` → return `Some(Command::Quit)`.

6. For state preservation across refresh (DiffParsed handler):
   - Before replacing diff_data, collect `collapsed_paths: HashSet<String>` by mapping each `NodeId::File(idx)` to `diff_data.files[idx].target_file.clone()`.
   - For `NodeId::Hunk(fi, hi)`, map to `(diff_data.files[fi].target_file.clone(), hi)`.
   - After replacing diff_data, rebuild collapsed set by finding new indices for the same paths.
   - Preserve `selected_index` by recording the current file path at that index, then finding it in the new data.

7. Remove the `run()` method entirely — the main loop moves to main.rs.
